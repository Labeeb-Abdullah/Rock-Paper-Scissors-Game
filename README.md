# Rock-Paper-Scissors-Game
I made this game to test my JavaScript skills, just after completing a javascript course 

A sleek, responsive, and minimalist Rock Paper Scissors game built from scratch to test and solidify my foundational vanilla JavaScript logic alongside modern HTML5 and semantic CSS3 styling.

This project marks the beginning of my web development journey as I master DOM manipulation and interactive user interfaces. More advanced applications and frontend projects are on the way!

Features:
Clean Logic: Pure JavaScript conditional routing for deterministic computer choices.

Minimalist UI: High-contrast aesthetic with smooth micro-interactions and hover effects.

Fully Responsive: Adapts seamlessly to mobile, tablet, and desktop screens using modern CSS Flexbox.
